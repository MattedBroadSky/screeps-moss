# Screeps Moss 项目

## 🎉 项目概述
**Screeps Moss** 是一个基于现代软件工程方法重新规划的高性能 Screeps AI 系统。项目采用模块化架构设计，专注于可维护性、扩展性和性能优化。

## 🏗️ 架构设计亮点

### 四层模块化架构
1. **应用层**: RoomManager (房间协调管理)
2. **业务层**: EnergyManager, CreepManager, RoleManager (核心业务逻辑)
3. **服务层**: ConfigService, MemoryService, LogService (基础服务)
4. **数据层**: 游戏状态、配置数据、历史数据

### 设计创新特点
- **智能身体配置**: Creep身体部件动态优化算法
- **能量经济模型**: 基于经济性的能量分配决策系统
- **分层压缩策略**: 内存数据的智能压缩管理
- **多级日志系统**: 结构化日志和智能分析

### 性能优化设计
- **CPU预算管理**: 每个模块有明确的CPU限制
- **内存优化**: 对象池、缓存、压缩策略
- **算法优化**: 复杂度控制，避免O(n²)算法
